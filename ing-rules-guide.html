-ing Spelling Rules

How to form the present participle (gerund) in English. The base rule is simple — the exceptions are where it gets interesting.

🟢 Base Rule Most verbs

✅ Just add -ing
The majority of English verbs simply take -ing at the end with no changes.
work → working play → playing eat → eating read → reading talk → talking

🟢 When NOT to double Important clarifications

📏 Two consonants at the end → don't double
When a verb ends in two consonants, the last consonant is never doubled.
help → helping work → working jump → jumping want → wanting watch → watching
📏 Two vowels before final consonant → don't double
When there are two vowels (a digraph) before the final consonant, the vowel is already "long" — no doubling needed.
rain → raining wait → waiting cook → cooking beat → beating read → reading

The doubling rule only applies with a short (single) vowel. Two vowels = long sound = no need to "protect" it.

📏 Stress on first syllable (multi-syllable) → don't double
In words with 2+ syllables, if the stress is not on the last syllable, don't double the final consonant.
open → opening listen → listening visit → visiting happen → happening offer → offering

Compare: beGIN → beginning (stress on last) vs LISten → listening (stress on first).

📏 Verbs ending in -y → just add -ing (always)
The -y never changes when adding -ing, regardless of what comes before it. (This is different from adding -ed where consonant+y → ied.)
play → playing study → studying carry → carrying enjoy → enjoying say → saying buy → buying

🟢 CVC Doubling Common pattern

✅ Double the final consonant + -ing
When a verb ends in Consonant-Vowel-Consonant (CVC) and the last syllable is stressed, double the final consonant.
run → running sit → sitting stop → stopping swim → swimming begin → beginning prefer → preferring

If stress is on the first syllable, don't double: open → opening, listen → listening, visit → visiting.

🔴 Exceptions Special cases

1️⃣ Silent -e → drop it
When a verb ends in a silent -e, remove it before adding -ing.
make → making come → coming dance → dancing write → writing
⚠️ Sub-exception: keep the -e (each has a different reason)
These verbs keep the -e, but not all for the same reason. Some are mandatory (would cause confusion), others are optional British preferences.
🚫 dye → dyeing (MANDATORY)
If you drop the -e, you get "dying" — which already exists from the verb "die" (to stop living). Keeping the -e is the ONLY way to distinguish them.
dye → dyeing ✅ (coloring fabric) die → dying ✅ (ceasing to live)

This is not optional. "Dyeing" and "dying" are two different words that must be spelled differently.

🚫 singe → singeing (MANDATORY)
If you drop the -e, you get "singing" — which already exists from "sing" (to make music). The -e separates them.
singe → singeing ✅ (light burning) sing → singing ✅ (making music)

Same as "dye/die" — ambiguity forces keeping the -e.

🇬🇧 age → ageing or aging (REGIONAL)
Both "ageing" (British) and "aging" (American) are correct. British English prefers keeping the -e to preserve the soft 'g' sound /dʒ/ visually, even though pronunciation doesn't change.
age → ageing 🇬🇧 age → aging 🇺🇸

Unlike "dye", there's no confusion risk — "aging" doesn't mean something else. It's a spelling preference, not a rule.

🔤 canoe → canoeing / queue → queueing (VOWEL ENDING)
These verbs end in a pronounced vowel -e (not a silent -e). The -e is part of the word's sound, so it stays. Dropping it would create an odd letter combination.
canoe → canoeing ✅ queue → queueing 🇬🇧 / queuing 🇺🇸 hoe → hoeing ✅ toe → toeing ✅ shoe → shoeing ✅

The -e in "canoe" is pronounced /uː/, not silent. Same logic as -ee words (see → seeing). "Queueing" vs "queuing" — both accepted, British prefers the -e.

2️⃣ Verbs ending in -ie → change to -y
When a verb ends in -ie, change it to -y before adding -ing. You can't have "-ieing" — it's an orthographic rule.
die → dying tie → tying lie → lying
3️⃣ Verbs ending in -c → add -k
When a verb ends in -c, add a -k before -ing to preserve the hard /k/ sound.
picnic → picnicking panic → panicking

Without the -k, the 'c' before -ing would sound like /s/ (as in "icing") instead of /k/.

4️⃣ Verbs ending in -w or -x → don't double
Even though they look like CVC patterns, -w and -x are phonetic exceptions — they never get doubled.
snow → snowing show → showing box → boxing fix → fixing mix → mixing

The -x represents two sounds /ks/, so phonetically the consonant is already "doubled". The -w after a vowel forms a diphthong, not a true consonant ending.

5️⃣ Verbs ending in -ee → just add -ing (don't drop e)
When a verb ends in double -ee, the final -e is NOT silent — it's a real vowel sound. Keep both and just add -ing.
see → seeing agree → agreeing free → freeing flee → fleeing guarantee → guaranteeing

The "drop silent -e" rule doesn't apply here because the -e in -ee is part of the vowel sound /iː/, not a silent letter.

6️⃣ -ge ending: when to keep vs drop -e
Most -ge verbs drop the -e normally. Only keep it when dropping would create ambiguity with another word (see singe → singeing above in sub-exceptions).
change → changing ✅ (drop -e) manage → managing ✅ (drop -e) charge → charging ✅ (drop -e) singe → singeing ✅ (keep -e, avoids "singing") tinge → tingeing ✅ (keep -e, optional)

The rule is simple: if dropping -e creates a word that already exists (sing/singing), keep it. Otherwise, drop it like normal.

7️⃣ British English: double -l even without stress
In British English, verbs ending in a vowel + l double the -l before -ing, even when the stress is NOT on the last syllable. American English does NOT double in these cases.
travel → travelling 🇬🇧 / traveling 🇺🇸 cancel → cancelling 🇬🇧 / canceling 🇺🇸 label → labelling 🇬🇧 / labeling 🇺🇸 fuel → fuelling 🇬🇧 / fueling 🇺🇸

Both spellings are correct — it depends on the dialect. If stress IS on the last syllable, both dialects double: compel → compelling, control → controlling.

8️⃣ Truly irregular forms
A few common verbs don't follow any predictable pattern — they just need to be memorized.
be → being ski → skiing taxi → taxiing

"be → being" keeps the -e (it's not silent, it IS the word). "ski → skiing" and "taxi → taxiing" just add -ing to the vowel ending directly.
