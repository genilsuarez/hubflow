Used To & Would

Talking about the past isn't just past simple — these forms show habits, states, and how comfortable you've become with something.

⏰ Used To vs Would — Past Habits

Works for states AND actions
used to
Any past habit or state that no longer exists.
I used to live in Rome. (state)
I used to play tennis. (action)
Only works for repeated actions
would
Repeated past actions — never states like be/have/like/know.
Every summer, we would visit our grandparents.
❌ The exception that trips everyone up
For STATES (be, have, like, know, own, believe) only used to works. You can never say "would be" or "would have" to describe a past state.
I used to be shy. ✓ I would be shy. ✗ She used to own a bakery. ✓

🔄 Used To / Be Used To / Get Used To

💡 Three different structures, three different meanings
used to + base verb = a past habit (no longer true).
be used to + noun/-ing = accustomed to something now (a settled state).
get used to + noun/-ing = the process of becoming accustomed (still adjusting).
I used to stay up late. (past habit) I'm used to the cold now. (accustomed) I'm getting used to waking up early. (adjusting)
